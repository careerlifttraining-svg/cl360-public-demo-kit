# CL360 Public Guidelines

## Use this repo for:
- Learning
- Demo content
- Podcast planning
- Public creative workflows
- Social media examples

## Do NOT include:
- Private CL360 prompt systems
- Paid frameworks
- Internal GPT instructions
- Artist private information
- Unapproved images
- Confidential business plans

## Creator Standards

Always use:

✅ Approved artist images  
✅ Accurate names  
✅ Respectful language  
✅ Culture-first storytelling  
✅ Real music. Real stories. Real culture.
